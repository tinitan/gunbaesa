청년들 건배사

(▶건배사로 돌아가기)

1. 그래도 : 그래 /내일은 / 도약할거야

2. 아우성 : 아름다운 / 우리들의 / 성공을 위하여

3. 명품백 : 명퇴조신 / 품위유지 / 백수방지

4. 무조건 : 무지힘들어도 / 조금만 참고 / 건승하자

5. 뚝배기 : 뚝심있게 /배짱있게 / 기운차게

6. 사이다 : 사고치지 말고 /이제 /다함께 가자 (다른뜻: 사랑을 / 이술잔에 담아 / 다함게 원샷)

7. 고감사 : 고생하셨습니다 / 감사합니다 / 사랑합니다

8. 상한가 : 상심하지 말고/ 한숨쉬지 말고/ 가슴을 펴자

9. 비행기 : 비젼을 가지고/ 행동하면 / 기적이 일어난다 (비상하자/ 행복을향해/ 기운차게)

10. 한우갈비 : 한마음인 우리를 갈수록 비상한다

11. 모바일 : 모든일이 / 바라는 대로 / 일어나라

12. 이기자 : 이런 / 기회를 / 자주 만들자

13. 비타민 : 비남말고 / 타협하여 / 민주적으로

14. 사우디 : 사나이 / 우정 / 뒤질때까지

15. 박카스 : 박력있고 / 카리스마 았고 / 스피드하게!

16. 청바지 : 청춘은 / 바로 / 지금부터

17. 재건축 : 재미있게 / 건강하게 / 축복하며 살자

18. 너나잘해 : 너와 / 나의 / 잘 나가는 새/ 해을 위하여

19. 스마일 : 스쳐도 웃고 /마주쳐도 웃고 / 일부러라도 웃자

20. 우생순 : 우리/ 생애 / 최고의 순간을 위하여

21. 우아미 : 우아하게 / 아름다운 / 미래을 위하여

22. 사우나 : 사랑과 / 우정을 / 나누자

(▶건배사로 돌아가기)
