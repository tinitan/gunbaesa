일반적 건배사

(▶건배사로 돌아가기)

1. 나이야 가라 : (나이는 숫자에 불과)

2. 자전거 : 자신감과 / 전문성을 갖고 / 거리로 나서자

3. 원더걸스 : 원하는 만큼/ 더도말고 / 걸러서 / 스스로 마시자

4. 노총각 : 노하지 말고/ 총대메지말고/ 가 세우지말고

5. 올버디 : 올해도 / 버팀목이 되고 / 디딤돌이 되자

6. 올보기 : 올해에도 / 보람있고 / 기분좋게 지냅시다

7. 올파 : 올해에도 / 파이팅 합시다

8. 고사리 : 고맙습니다 / 사랑합니다 / 이해합니다

9. 빠삐용 : 빠지지 말고 / 삐지지 말고 / 용서하면서 살자

10. 쾌쾌쾌 : 유쾌 / 상쾌 / 통쾌

11. 하하하 : 하루에 세번 하하하 웃고 살자
